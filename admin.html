Export Listings
Export your data to share with the customer portal or keep a backup. For Netlify/GitHub hosting, export as JSON, replace the data in your index file.

How to Publish to GitHub Pages

  1. Add your listings using the Listings tab above.
  2. Click Export JSON — a file will download.
  3. Open your customer.html file and find the script section.
  4. Replace the localStorage default or embed the JSON as a defaultListings variable.
  5. Push to GitHub → your customer portal auto-updates!
  6. Customers visiting your site will see all properties instantly — no login needed.
